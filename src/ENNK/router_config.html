ENNK Router Config

set encrypted privileged EXEC mode password

  • R2(config)#enable secret c1sco1234

Set the domain name to ccna-lab.com

  • R2(config)#ip domain-name ccna-lab.com

Disable DNS lookup

  • R2(config)#no ip domain lookup

encrypt plain text password

  • R2(config)#service password-encryption

Configure the username SSHadmin with an encrypted password of 55Hadm!n

  • R2(config)#username SSHadmin secret 55Hadm!n

Generate a set of crypto keys

  • R2(config)#crypto key generate rsa

Assign cisco as the console password

  • R2(config)# line console 0
  • R2(config-line)#password cisco

configure sessions to disconnect after six minutes of inactivity

  • R2(config-line)#exec-timeout 6 0

To prevent console messages from interrupting commands, use the logging synchronous command

  • R2(config-line)#logging synchronous

enable login

  • R2(config-line)#login

Assign cisco as the vty password

  • R2(config)# line vty 0 4
  • R2(config-line)# password cisco

configure the vty lines to accept SSH connections only

  • R2(config-line)# transport input ssh

configure sessions to disconnect after six minutes of inactivity

  • R2(config-line)# exec-timeout 6 0

enable login using the local database

  • R2(config-line)# login local

Create a banner

  • R2(config)# banner motd $ WARNING Authorized Users Only! $

Enable IPv6 Routing

  • R2(config)# ipv6 unicast-routing
