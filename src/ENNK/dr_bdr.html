DR and BDR

Verify OSPF neighbor states

  • RA# show ip ospf neighbor

Turn on ip OSPF adjacency debugging

  • RA# debug ip ospf adj

Configure OSPF priorities on router

  • RA(config)# interface g0/0
  • RA(config-if)# ip ospf priority 200

Force an election by resetting the OSPF process on the router

  • RA# clear ip ospf process
