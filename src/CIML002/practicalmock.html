Practical Mock

Do you know how to do the following
  1. Launch Control Panel

    Ans: Search > Control Panel

  2. Launch Administrative Tools

    Ans: Search > Administrative tools

  3. Rename the “Play” grouping to “Games” and resize it

    Ans: At Windows start, left click on "Play" and edit to "Games"

    Right click on tile, click on resize.

  4. With one click of the mouse, return back to the desktop

    Ans: left click on task view on taskbar

  5. Add a quick launch button for command prompt

    Ans: Right click on command prompt app -> pin to task bar

  6. Add a quick launch button that automatically runs command prompt as administrator

    Ans: Right click command prompt on taskbar, right click command prompt, right click propeties.

    click on checkbox: run as administrator

  7. Pin an application to “Start Menu"

    Ans: Right click on application -> pin to start

  8. Change the system time

    Ans: Search -> date and time, slide set time automatically -> off

    under set date and time manually -> click change

  9. Configure the notification area to show all icons

    Ans: Search task bar setings

    Click on "select which icons appear on the taskbar"

    Select which icons appear on the taskbar

    Slide "Always show all icons in the notification area" to on

  10. Launch System Settings with just two clicks of the mouse

    Ans: click start -> click cog wheel icon (settings)

  11. Autohide the taskbar

    Ans: Right click on taskbar, select task bar settings

    slide "automatically hide task bar in desktop mode" to on

  12. View the system resource (eg, CPU/memory/disk) utilisation and background processes

    Ans: Start -> search task manager

  13. Change your password

    Ans: start -> settings -> accounts -> sign in options

    under password -> change

  14. Start/stop background processes and configure their startup status

    Ans: Start task manager.

    Right click on background process to stop

    File -> Run new task -> name of new task to start

    Startup tab -> right click on task -> select enable or disable

  15. Launch multiple desktops and learn to navigate among the various desktops

    Ans: On Task bar -> Task view -> + new desktop -> switch between desktops

    Click on x to close desktop

  16. What will be the shortcut key in windows to perform cut operation?

    Ans: Ctrl+X

  17. Zip files are _____.

    Ans: Compressed Files

  18. Which of the following is the file directly used by Operating System?

    Ans: .ini

  19. In Windows operating system it is a very important part of Windows, clicking on it opens up what is called the start menu

    Ans: Start button

  20. In Windows operating system the ______ is the main screen area that one sees after he turn on a computer and log on to Windows.

    Ans: Desktop

  21. In Windows operating system these are small pictures that represent files, folders, programs, and other items. Each one is used to make the computer do something.

    Ans: Icons

  22. Microsoft Windows provides a graphics application named _____

    Ans: Paint

  23. In Windows operating system it is the long horizontal bar at the bottom of a screen. Unlike the desktop, which can get obscured by the windows on top of it, it is visible almost all the time.

    Ans: Taskbar

  24. Text Editor that is the part of Windows operating system

    Ans: NotePad

  25. How can you find the amount of memory installed on your computer?

    Ans: Use Control Panel

  26. Which keyboard shortcut enables you to select the contents of an entire document?

    Ans: Ctrl+A

  27. Which keyboard shortcut enables you to save in any Windows application?

    Ans: Alt+S

  28. When you let your pointer linger over an object, the pointer is _____.

    Ans: Hovering (Ans)

  29. The Windows 10 Store app is linked to which of the following?

    Ans: The user’s Microsoft account

  30. What is the keyboard shortcut to lock your computer?

    Ans: Windows Logo+L
