  16. Compare Application Software and System Software.
    17. SYSTEM SOFTWARE APPLICATION SOFTWARE
    The software which provides a platform for the user to interact with the hardware of a computer are known as system softwares Those softwares which run on a operating system (OS is a system software) serving specific purpose are called application softwares
    System softwares are needed to run application softwares Application softwares are not needed to run system softwares
    Runs in the background and act as a platform Runs in the background and interacts with the user
