  1. The smallest unit of information a computer can understand and process is known as a __

    Ans: bit

  2. How many options does a binary choice offer?

    Ans: Two

  3. A string of eight 0s and 1s is called a ______

    Ans: Byte

  4. A computer works on a _______ number system.

    Ans: Binary

  5. Which of the following is not a binary number?

    Ans: 102

  6. Conversion of binary number (1001001)2 to hexadecimal is ____.

    Ans: 49

  7. Conversion of binary number (101110)2 to hexadecimal is ____.

    Ans: 2E

  8. Information on a computer is stored as ______

    Ans: Digital data

  9. The basic goal of computer process is to convert the data into ______.

    Ans: information

  10. Computer uses the ______ number system to store data and perform calculations.

    Ans: binary

  11. Which of the following is hexadecimal number equivalent to binary number (1111 1001)?

    Ans: F9

  12. 1010 equivalent decimal number is
