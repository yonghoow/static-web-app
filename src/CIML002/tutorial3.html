CIML002 Tutoral 3

  1. List the 4 essential managers of an operating system

    Ans: Memory, process, device, file managers

  2. When an operating system has networking capability, what is the additional fifth essential manager it must have?

    Ans: Network manager

  3. What are the tasks that each of the managers of an operating system will perform?

    Ans: Process manager - allocating CPU resources to processes.

    File manager - manages common operations (create, copy, etc) on a file

    Device manager - monitors all hardware peripheral devices attached to a computer.

    Memory manager - performs memory (RAM) management to programs

  4. What is the computer hardware component that would be managed by the memory manager, and processor management respectively?

    Ans: Memory manager manages RAM. Processor allocates CPU to a process

  5. Differentiate between a job and a process

    Ans: A process is a program in execution. A job is a complete unit of work under execution. A job can consist of many processes

  6. What is a processor manager made up of?

    Ans: Process manager involves creating, scheduling, managing deadlock, and terminating processes

  7. Illustrate the 5 job/process states in a typical job and state which scheduler (job or process) handle it.

    Ans: Hold, run. wait, ready, finish. Process scheduler handles it.

  8. What is a Process Control Block (PCB)?

    Ans: A process control block is maintained for a process throughout its lifetime, and is deleted once the process terminates

  9. Explain what the difference between preemptive and non-preemptive scheduling policy.

    Ans: Under non-preemptive scheduling, once a CPU has been allocated to a process, it cannot be interrupted until it finishes or terminates

    With preemptive, a process can be interrupted during its course of action

  10. List the 4 process scheduling algorithms and state if it is preemptive or non-preemptive.

    Ans: FIrst come, first serve - non-preemptive

    Shortest next job - non-preemptive

    shortest remaining time - pre-emptive

    round robin - preemptive

  11. Given the 3 jobs and their corresponding CPU cycle or run time in milliseconds as shown in the below table, calculate their average turn-around time, based on
    Job: A B C
    CPU cycles (ms) 8 3 5
    • First-come, first-served (FCFS) process scheduling algorithm, assuming Job A arrives first, followed by Job B, and then Job C.
    • First-come, first-served (FCFS) process scheduling algorithm, assuming Job C arrives first, followed by Job B, and then Job A.
    • Shortest job next (SJN) process scheduling algorithm

    Ans: FCFS (A, B, C). Ave turn-around time = (0 + 8 + 3) / 3 = 3.66 ms

    FCFS (C, B, A). Ave turn-around time = (0 + 5 + 3) / 3 = 2.66 ms

    SJN (B, C, A). Ave turn-around time = (0 + 3 + 5) / 3 = 2.66 ms

  12. Using Shortest Remaining Time (SRT) preemptive process scheduling algorithm, derive their average turn-around time, as they arrive in the sequence as shown

    Job A B C D
    CPU cycle (ms) 6 3 1 4

    Ans: SRT (A,B,C,B,D,A). Ave turn-around time = (9 + 2 + 3 + 5) / 4 = 4.75 ms
